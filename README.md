# 🦷 Smart Clinic

نظام متكامل لإدارة عيادات الأسنان والمراكز الطبية — يسهّل إدارة الجوانب الطبية والإدارية والحسابية بشكل فعّال.

## المميزات الرئيسية

- 🦷 **Dental Chart**: تحديد الجزء المصاب من السن وإضافة إجراء (حشو عصب/خلع/تنظيف) مع إضافة السعر تلقائياً على الكشف.
- 🔁 **الباقات والتذكيرات**: باقات متعددة الجلسات مع متابعة أوتوماتيك و Reminders لتأكيد/تغيير المواعيد.
- 📦 **ربط المخزون**: ربط منتج بأكثر من إجراء مع سحب تلقائي من الكمية.
- 🤖 **واتساب بوت**: تتبع الرسائل، إرسال ملفات (فيديو/صور/PDF)، وجدولة الرسائل مع التكرار.
- 🧠 **المساعد الذكي**: على علم بملف المريض، أوامر صوتية، إنشاء روشتات وتقارير طبية.
- 📅 **حجز إلكتروني**: حجز المرضى مواعيدهم عبر الإنترنت.
- ⏱️ **تنظيم المواعيد**: تقويم شهري لكل طبيب وفرع منفصل.
- 🧑🏻‍⚕️ **بروفايل المريض الكامل**: كل التاريخ (تشخيصات، روشتات، حجوزات، مدفوعات، ملفات، شكاوي).
- 📊 **التقارير والملفات** + ☁️ **نظام سحابي** يعمل من أي جهاز.
