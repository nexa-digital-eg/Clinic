# 🦷 Smart Clinic

نظام متكامل لإدارة عيادات الأسنان والمراكز الطبية — يسهّل إدارة الجوانب الطبية والإدارية والحسابية بشكل فعّال.

## المميزات الرئيسية

- 🦷 **Dental Chart**: تحديد الجزء المصاب من السن وإضافة إجراء (حشو عصب/خلع/تنظيف) مع إضافة السعر تلقائياً على الكشف.
- 🔁 **الباقات والتذكيرات**: باقات متعددة الجلسات مع متابعة أوتوماتيك و Reminders لتأكيد/تغيير المواعيد.
- 📦 **ربط المخزون**: ربط منتج بأكثر من إجراء مع سحب تلقائي من الكمية.
- 🤖 **واتساب بوت**: تتبع الرسائل، إرسال ملفات (فيديو/صور/PDF)، وجدولة الرسائل مع التكرار.
- 🧠 **المساعد الذكي**: على علم بملف المريض، أوامر صوتية، إنشاء روشتات وتقارير طبية.
- 📅 **حجز إلكتروني**: حجز المرضى مواعيدهم عبر الإنترنت.
- ⏱️ **تنظيم المواعيد**: تقويم شهري لكل طبيب وفرع منفصل.
- 🧑🏻‍⚕️ **بروفايل المريض الكامل**: كل التاريخ (تشخيصات، روشتات، حجوزات، مدفوعات، ملفات، شكاوي).
- 📊 **التقارير والملفات** + ☁️ **نظام سحابي** يعمل من أي جهاز.

## الحالة الحالية

✅ **اكتملت كل المراحل (0–9)**. راجع **[الخطة المعمارية الكاملة](docs/PLAN.md)**.

- **الأساس**: Next.js + TypeScript + Tailwind (عربي RTL) + PostgreSQL/Prisma + مصادقة وأدوار.
- **المرضى**: بروفايل كامل (تشخيصات/روشتات/حجوزات/مدفوعات/ملفات/شكاوي/أسنان).
- **الحجوزات**: تقويم شهري لكل طبيب/فرع + حجز إلكتروني عام.
- **Dental Chart**: مخطط أسنان تفاعلي + سعر الإجراء تلقائياً على الكشف.
- **الفوترة**: فواتير ومدفوعات وأرصدة متّسقة.
- **المخزون**: منتجات + ربط بالإجراءات + سحب تلقائي.
- **الباقات والتذكيرات**: متابعة أوتوماتيك + تأكيد/تغيير الموعد.
- **واتساب**: إرسال/جدولة/تكرار/تتبّع (محاكاة + حقيقي).
- **المساعد الذكي**: دردشة بسياق المريض + أوامر صوتية + روشتات + تقارير (محاكاة + حقيقي).
- **التقارير + السحابة**: لوحة تقارير + Docker + PWA.

> المميزات الخارجية (واتساب/المساعد الذكي) تعمل في **وضع محاكاة** بدون مفاتيح،
> وتتحوّل للوضع الحقيقي فور إضافة المفاتيح في `.env` دون أي تعديل كود.

## التشغيل محلياً

```bash
# 1) ثبّت الحزم
npm install

# 2) جهّز ملف البيئة
cp .env.example .env   # عدّل DATABASE_URL و AUTH_SECRET

# 3) شغّل قاعدة البيانات (Docker) أو استخدم PostgreSQL محلي
npm run db:up

# 4) أنشئ الجداول وأضف بيانات تجريبية
npm run db:migrate
npm run db:seed

# 5) شغّل التطبيق
npm run dev
```

ثم افتح http://localhost:3000 وسجّل الدخول بـ:
`admin@clinic.com` / `admin123`

للنشر السحابي راجع **[دليل النشر](docs/DEPLOYMENT.md)** (Docker / Vercel).
